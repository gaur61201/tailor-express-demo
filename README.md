# Tailor Express — Demo Website

Demo website for Tailor Express, a 9-branch tailoring service
in Kuwait.

## Live demo
[gaur61201.github.io/tailor-express-demo](https://gaur61201.github.io/tailor-express-demo/)

## Status

- GSAP + ScrollTrigger for animations
- Three.js for 3D Gallery section
- Bilingual EN/AR (RTL support)
